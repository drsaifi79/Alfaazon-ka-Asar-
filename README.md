# Alfaazon-ka-Asar-
Hindi/ Urdu Shayri, हिंदी/उर्दू शायरी 
जब तू "तारीफ" करती है मेरी, तो असर होता है "दिन की तारीख तक" 
और जब तू "बुराई" करती है, तो असर होता है "दिल की गहराई तक" 

1
"कैसा इल्जाम दिया है तुमने, मेरी मोहब्बत को बेरुखी सजा बात कर," 
खुद को समझा मेरे दोस्त, परेशां ना हो बहारों को फिजा बात कर"...

यह मेरा पहली शायरी थी जो मैंने 1994 में कही थी और उसके बाद, बस में लिखता गया... लिखता गया... और लिखता चला गया ....

2
इस लावारिस दिल में, नाम है तेरा तस्वीर है तेरी,
       तू चाहे कुछ भी है, पर तकदीर है मेरी ...

3.
सही से प्यार का इकरार किया, ना इनकार किया .
फिर यह कैसा जुल्म किया ....
मारकर पत्थर तूने दिल पर, क्या एहसास खुशी का किया ....
